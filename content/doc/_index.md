# 中山大学智能工程学院本科生课程共享计划

本项目旨在整理和分享中山大学智能工程学院本科生课程的复习策略、选课建议、期末报告等各种资料，供学弟学妹们参考学习。想要打破大家的信息差，帮助一些不会获取信息，社恐的同学能够获得足够的学长学姐的帮助。

## 文件架构

左侧可见，本项目的文件架构为：

- 大一公共课程
- 往届课程
- 智慧交通（交通工程）
- 智能科学与技术
- 全部课程复习策略整理

由于智交和智科在大一的课程差异不大，因此将两个专业的大一课程单独列出来。同时因为智工近年课程调整较大，有的课程可能已经废弃，储存在往届课程中，有的课程调整了所在学期，目前本项目以维护者所在的 **24 级**组织，如果未来有新的变化，可以根据新一届学生的课程设置组织。也欢迎同学们前往[贡献方式](/about/)中提 Issue 来告知维护者每届的课程变化。

全部课程复习策略整理是将所有复习资料汇总而来，以防有人只需要阅读复习建议。

## 涵盖内容及贡献范围

本项目目标是涵盖智能工程学院开设的所有课程内容的**复习资料、课程简介、复习建议和选课建议**。目前集齐的比较多的是大部分课程的复习资料，但是课程简介、复习建议和选课建议这几个比较缺乏，因此如果你有**任何想提交的课程简介、复习建议和选课建议**，欢迎到[贡献方式](/about/)中通过 PR 或者邮箱的方式来贡献。

总体来说，我们欢迎：

- **往年真题与复习资料**：为已有资料的课程补充带答案的历年真题，或提交高质量的复习笔记与总结；
- **空白课程的资料填充**：为尚未收录或资料匮乏的课程提交基础资料，帮助完善课程覆盖范围；
- **课程评价与建议**：提交尽量客观的课程简介、复习建议或选课建议，建议附上昵称与 GitHub 账号以便署名与追溯。

我们不欢迎：

- **带有明显个人情绪或主观偏见的评价**：包括对教师的贬义描述、情绪化表达或人身攻击。选课建议应基于客观的课程特点、考核方式、作业难度、工作量等事实性描述，而非个人好恶；
- **大量来源不明的题目或答案**：无法确认来源的习题、考题、答案合集，由于不同学校考试范围和难度差别巨大，我们不是特别鼓励这部分内容的大量提交。

### **经验分享与答疑**

我们同样欢迎高年级同学或校友以[贡献者](/about/)身份分享个人经历与见解——无论是学业规划、保研或出国的经验，还是 AI 时代下的技能转型思考，都是对低年级同学极具参考价值的内容。

## 温馨提示

1. **本项目由学生自发组织，不代表学院意志**；
2. 本项目的下载默认使用的是[国内加速](ghfast.top)的方式下载，如果屡次失败可能是该网站挂了，这种情况下可以点击右侧 GitHub 剪影前往对应的仓库下载；
3. 左栏的一些课程每届可能名字略有不同，或者课程所在的学期不同，可以右上角搜索去找或者翻一翻别的学期的文件夹，也许在那里；
4. 往届课程是过去开设过，可能目前不开设的课程。

## 重要提醒

本项目中的资料仅供参考，请同学们独立完成作业和实验，遵守学术诚信。

> **第三十六条** 考试过程中或考试结束后发现下列行为之一的，应当认定相关的考生实施了考试作弊行为，视情节轻重和考生认错态度，给予下列处分：
>
> （二）评卷过程中被认定为答案雷同的，或课程论文被认定有抄袭行为的，给予记过或留校察看处分；
>
> （五）其他应认定为考试作弊的行为，视情节轻重，给予相应处分；
>
> （六）两次考试作弊者，给予开除学籍处分。
