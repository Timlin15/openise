      中山大学智能工程学院本科生课程作业

      中山大学智能工程学院本科生课程作业

      受清华大学、浙江大学相关项目启发，创立了本项目。

      由于本仓库未启用 git-lfs，因此请不要上传单个超过 100M 的文件，否则 commit 无效。

      下载单个文件夹

      若要下载单个文件夹，复制该文件夹的网址，粘贴入 DownGit 中，选择 download 即可。

      例如，要下载"数字逻辑"文件夹，复制该文件夹的 GitHub 网址（如 https://github.com/changyicheng1234/sysu-ise-course-materials/tree/main/数字逻辑），粘贴到 DownGit 中即可下载。

      前言

      本项目旨在整理和分享中山大学智能工程学院本科生课程的复习策略、选课建议、期末报告等各种资料，供学弟学妹们参考学习。想要打破大家的信息差，帮助一些不会获取信息，社恐的同学能够获得足够的学长学姐的帮助。

      重要提醒：本项目中的资料仅供参考，请同学们独立完成作业和实验，遵守学术诚信。若发现任何侵权内容，请在 issues 下指出，非常感谢。

      ⭐ 如果这个项目对你有帮助，欢迎点个 Star 和 Watch（关注）支持一下！（在界面右上角） 你的 Star 是我们持续维护的最大动力，也能让更多同学发现这个项目。关注后有新资料更新时会第一时间收到通知~
